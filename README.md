# GitHub API Caller
 This service provides endpoints to list repositories, fetch commits, manage issues, and create pull requests, all wrapped in a clean, typed interface utilizing `Pydantic` and `HTTPX`.

## 📚 Documentation

Detailed documentation has been separated into specific guides to help you get started quickly and reference API details. Please refer to the following documents in the `app/docs/` directory:

- [**Setup Guide & Quick Start**](docs/setup.md): Instructions on how to set up your environment, configure secrets (`GH_TOKEN`), and safely run the FastAPI server.
- [**API Endpoints Documentation**](docs/api_endpoints.md): Comprehensive documentation outlining the available `/api/v1/github/` routes, their expected payloads, and query parameters.
